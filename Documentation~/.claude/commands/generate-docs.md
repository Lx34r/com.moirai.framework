# 生成文档

为指定的代码生成详细的文档。

## 参数
- $TARGET: 要生成文档的文件或目录路径
- $FORMAT: 文档格式（可选：markdown, xml, html）

## 文档类型

### 1. API 文档
- 类和方法说明
- 参数和返回值
- 使用示例
- 异常说明

### 2. 架构文档
- 系统架构图
- 模块关系图
- 数据流图
- 依赖关系

### 3. 用户指南
- 快速开始
- 功能说明
- 配置说明
- 常见问题

### 4. 开发指南
- 编码规范
- 设计模式
- 最佳实践
- 贡献指南

## 文档生成流程

### 1. 代码分析
- 解析代码结构
- 提取注释和文档字符串
- 分析依赖关系

### 2. 内容生成
- 生成文档内容
- 添加示例代码
- 创建图表

### 3. 格式化
- 应用文档模板
- 添加样式
- 生成目录
