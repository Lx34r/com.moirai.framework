# 解释代码

详细解释指定代码的功能、设计和实现细节。

## 参数
- $CODE: 要解释的代码片段或文件路径

## 解释维度

### 1. 功能概述
- 代码的主要功能
- 解决的问题
- 使用场景

### 2. 代码结构
- 类和方法的组织
- 数据流向
- 控制流程

### 3. 设计模式
- 使用的设计模式
- 设计决策的原因
- 优缺点分析

### 4. 关键实现
- 核心算法
- 重要逻辑
- 特殊处理

### 5. 依赖关系
- 外部依赖
- 内部依赖
- 依赖注入

### 6. 使用示例
- 如何调用
- 参数说明
- 返回值说明
